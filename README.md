# Skill Mentor Backend

Skill Mentor Backend is the server-side application designed to manage and support the operations of the Skill Mentor platform. Built entirely in Java, this backend provides the necessary APIs and infrastructure for seamless communication between the front-end client and the database.

## Key Features

- **User Management**: Handle user registration, authentication, and authorization.
- **Skill Catalog Management**: Manage and update the catalog of skills available for mentorship.
- **Mentorship Matching**: Match mentors with mentees based on skills and interests.
- **Session Scheduling**: Support scheduling of mentorship sessions with reminders and notifications.
- **Analytics and Insights**: Provide insights into mentorship program performance with data tracking.

## Technology Stack

- **Programming Language**: Java 
- **Frameworks**: (Springboot)
- **Database**: (Postgrey SQL)
- **ORM**: Hibernate
- **Authentication & Authorization** - clerk

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sithija-sulochana/Skill-mentor-backend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Skill-mentor-backend
   ```
3. Build the project:
   ```bash
   ./mvnw clean install
   ```
4. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
